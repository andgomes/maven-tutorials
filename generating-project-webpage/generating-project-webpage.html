Gerando a Página Web de um Projeto

Um dos diferenciais do Maven é a possibilidade de utilizar plugins para os mais diversos tipos de funcionalidades, que vão além de builds relacionados somente com a aplicação em si. Por esse motivo, o Maven não é considerado apenas uma ferramenta de build, mas uma ferramenta de gerenciamento de projetos. Neste tutorial, veremos como gerar uma página HTML com a documentação básica de um projeto Maven através do plugin Site, o plugin que é utilizado quando invocamos a fase site do lifecycle Site.

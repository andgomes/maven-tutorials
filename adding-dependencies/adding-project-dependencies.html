Adicionando Dependências

Com exceção de programas muito simples, a maioria dos projetos Java utiliza APIs externas. Essas APIs, que normalmente estão no formato JAR, são chamadas de dependências e devem estar disponíveis para que os programas possam utilizá-las. Em projetos que não utilizam nenhuma ferramenta de build, essas APIs geralmente são armazenadas em um diretório separado e são referenciadas na compilação dos códigos-fonte. Nos projetos Maven, como era de se esperar, todas as dependências são declaradas no arquivo POM.

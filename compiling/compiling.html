Compilando com o Plugin Compiler do Maven

Uma das primeiras ações a executar depois de termos escrito o código de uma classe Java é compilá-lo. Neste tutorial irei mostrar como compilar o código-fonte do diretório main/java/ com o plugin Compiler do Maven.

Antes de executar o comando para compilar, é interessante saber que, por padrão, o Compiler assume que o seu código-fonte está escrito de acordo com a versão 5 do Java, ou seja, se seu código tiver lambda expressions(da versão 8) ou apenas o diamond operator(da versão 7), o plugin Compiler lançará um erro informando que seu código não é compatível com a versão 5.

Para alterar a versão do código-fonte(source) e a versão do arquivo compilado(target), altere o arquivo pom.xml como mostrado abaixo.

<build>
--<plugins>
----<plugin>
------<groupId>org.apache.maven.plugins</groupId>
------<artifactId>maven-compiler-plugin</artifactId>
------<version>3.5.1</version>
------<configuration>
--------<source>1.7</source>
--------<target>1.7</target>
------</configuration>
----</plugin>
--</plugins>
</build>

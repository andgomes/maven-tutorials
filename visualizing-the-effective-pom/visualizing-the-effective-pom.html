Visualizando o Effective POM de um Projeto

Todo projeto Maven possui um arquivo POM que contém as informações e as configurações que serão utilizadas durante o build. Porém, somente com as informações do arquivo POM de um projeto, o Maven não tem acesso à outras informações importantes, como a localização dos códigos-fonte, onde salvar as classes compiladas, etc.

Para acessar essas informações, o Maven utiliza um arquivo chamado pom-4.0.0.xml, conhecido como Super POM, que fica localizado no diretório M3_HOME/lib/(M3_HOME é o diretório de instalação do Maven), dentro do arquivo JAR maven-model-builder.3.3.9.jar.

Quando surgem erros no build que são difíceis de encontrar, podemos recorrer ao Effective POM, que é o resultado da junção do Super POM com o arquivo POM específico do projeto. Para visualizar o Effective POM, vamos executar o goal effective-pom do plugin Help.

$ mvn help:effective-pom

O Maven irá exibir o resultado no terminal. Como o arquivo gerado é muito extenso, algumas partes foram omitidas.

Visualizando o Effective POM de um Projeto

Sabemos que todo projeto Maven possui um arquivo POM que é utilizado para armazenar todas as configurações do projeto. Porém, somente com as informações do arquivo POM de um projeto o Maven não tem como saber outras informações importantes, como localização dos códigos-fonte, onde salvar as classes compiladas.

Para acessar essas informações, o Maven utiliza um arquivo chamado pom-4.0.0.xml, conhecido como Super POM, que fica localizado em M3_HOME/lib/, dentro do arquivo JAR maven-model-builder.3.3.9.jar.

Quando surgem erros no build que são difíceis de encontrar, podemos recorrer ao Effective POM, que é a junção do Super POM com o arquivo POM específico do projeto. Para visualizar o Effective POM, vamos executar o goal effective-pom do plugin Help.

$ mvn help:effective-pom

O Maven irá exibir, no terminal, o resultado da junção do Super POM com o POM do projeto.

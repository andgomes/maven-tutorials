Limpando diretórios adicionais com o Plugin Clean

Nos posts anteriores vimos várias formas de limpar o diretório target, o diretório que é criado quando fazemos um build do projeto. A grande maioria dos projetos Maven só precisam de um diretório para armazenar todos os artefatos gerados no build, mas há a possibilidade de um projeto executar um script que crie um diretório adicional, além do target. O plugin Clean nos dá opção de listar quais diretórios e arquivos queremos excluir ao executar o goal clean. Neste post, veremos como fazer isso.

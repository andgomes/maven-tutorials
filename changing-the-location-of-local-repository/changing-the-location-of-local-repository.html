Alterando a Localização do Repositório Local

Quando executamos algum comando do Maven, alguns plugins são baixados do repositório central. Esses plugins ficam armazenados no diretório /home/username/.m2, que é utilizado como repositório local padrão.

Como o diretório .m2 é um diretório oculto, quando queremos ver seu conteúdo precisamos digitar as teclas ctrl+h. Então para facilitar o acesso, podemos querer utilizar outro diretório como repositório local.

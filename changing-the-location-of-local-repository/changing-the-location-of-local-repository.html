Alterando a Localização do Repositório Local

Quando executamos algum comando do Maven, alguns plugins são baixados do repositório central. Esses plugins ficam armazenados no diretório /home/username/.m2, que é utilizado como repositório local padrão.

Como o diretório .m2 é um diretório oculto, quando queremos ver seu conteúdo precisamos digitar as teclas ctrl+h. Para facilitar o acesso, podemos querer utilizar outro diretório como repositório local.

Vamos alterar o arquivo /home/username/.m2/settings.xml para indicar o caminho do diretório que será utilizado. Caso o arquivo não exista, crie um.

settings.xml

<settings xmlns="http://maven.apache.org/SETTINGS/1.0.0"
----------xmlns.xsi="http://www.w3.org/2001/XMLSchema-instance"
----------xsi:schemaLocation="http://maven.apache.org/SETTINGS/1.0.0
----------http://maven.apache.org/xsd/settings-1.0.0.xsd">
--...
--<localRepository>/home/username/maven-repo</localRepository>
--...
</settings>

Agora quando executarmos um comando, o Maven irá baixar os plugins para o diretório /home/username/maven-repo.

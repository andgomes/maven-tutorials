Tutorial de Instalação do Apache Maven no GNU/Linux

Neste tutorial, irei apresentar os passos necessários para instalar a última versão do Apache Maven(atualmente, a 3.3).

Há pelo menos duas formas possíveis de instalar o Maven: através do programa apt do GNU/Linux ou configurando a variável de ambiente PATH. Com o programa apt, só há um passo necessário: executar o comando apt-get install mvn. Porém, o usuário não tem tanto controle sobre a localização dos diretórios do Maven, ao contrário da instalação manual, que foi a que eu escolhi para esse tutorial.

Antes de qualquer coisa, temos que fazer o download do Maven no site oficial da Apache Foundation.

Na seção Files, clique no link do arquivo apache-maven-3.3.9-bin.tar.gz e aguarde o download.

Para extrair o arquivo tar.gz, primeiro abra o Terminal e entre no diretório onde o arquivo foi baixado com o comando cd diretório-do-arquivo. Por exemplo, se o arquivo estiver no diretório de Downloads, o comando será cd ~/Downloads.

Em seguida, execute o comando tar -zxfv apache-maven-3.3.9-bin.tar.gz. Pode ser necessário instalar o programa tar, você pode instalar através do utilitário apt do GNU/Linux, com o comando apt-get install tar.

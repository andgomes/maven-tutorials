Adicionando os Javados no Site de um Projeto

Nos últimos dois posts, vimos como gerar as páginas do site de um projeto com o plugin Site e como gerar os Javadocs das classes com o plugin Javadoc. Neste tutorial, vamos utilizar os dois plugins em conjunto para adicionar a documentação das classes no site do projeto.

Por conveniência, vamos utilizar o mesmo projeto do último post, o string-int-math, que possui os comentários Javadoc na classe e nos métodos. Vamos alterar apenas o arquivo POM e configurar os plugins de forma que, quando invocarmos a fase site, os javadocs estejam disponíveis no site do projeto.
